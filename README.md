# Hospital Emergency Room Analysis Dashboard | MIS Project

**End-to-End MIS Dashboard** built to monitor and improve Hospital Emergency Room operations.

![Dashboard Screenshot](Screenshots/dashboard_full_view.png)

## 🎯 Project Objective
To create an interactive dashboard that helps hospital administrators track patient inflow, reduce wait times, improve patient satisfaction, and optimize resource allocation.

## 📊 Key Features & KPIs Delivered

- **Total Patients**: 513 (Monthly)
- **Average Wait Time**: 36.32 minutes
- **Average Satisfaction Score**: 4.96/10
- **Admission Rate**: 52.44% admitted
- **Timeliness Analysis**: On-time vs Delayed patients
- **Demographics**: Age group, Gender, Race distribution
- **Department Referrals** analysis

## 🛠️ Tools & Technologies
- **Microsoft Excel** (Advanced)
- PivotTables & Pivot Charts
- Slicers & Dashboard Layout
- Conditional Formatting
- Sparkline Trends (Daily)
- Data Cleaning & Transformation
